## Developer Notes

### Structure
This repository is a monorepo-style collection. Most applications are self-contained within their respective directories.

*   **Backend:** typically found in `backend/` or `src/main/java` within project folders.
*   **Frontend:** typically found in `frontend/` or `src/` within project folders.

### Getting Started
To work on a specific app, navigate to its directory:
```bash
cd <app-directory-name>
```
Refer to the `README.md` within each subdirectory for specific installation and usage instructions.

### Technologies
Common technologies across this collection include:
*   **Java / Spring Boot** (Backends)
*   **React / TypeScript** (Frontends)
*   **Node.js**
*   **PostgreSQL**
