# Tech Career Guidance Web Application

## Project Overview

This web application provides clear, actionable pathways into technology careers for individuals with diverse backgrounds. It features personalized career recommendations, 1-year learning plans, and community support in a modern, clean dark mode design with mobile-first approach.

## Key Features

### User-Centric Design
- Modern, clean dark mode interface
- Mobile-first responsive design
- Accessible for users with varying technical literacy
- Optimized for limited connectivity

### Personalized Assessment
- Interest & passion exploration questionnaires
- Skill & strength assessment tools
- Work style & preferences evaluation
- Job role interest selection
- Technological access assessment

### Career Recommendation
- Algorithm-driven career matching
- Detailed job role descriptions
- Salary ranges and market demand information
- Personalized explanation of recommendations

### Learning & Development
- 1-Year structured learning plans
- Weekly/monthly milestones
- Personalized resource recommendations
- Progress tracking tools
- Portfolio building guidance
- Networking strategies
- Interview preparation resources
- Resume building tools
- Mentorship matching

### Community Support
- Discussion forums by topic
- Expert Q&A sessions
- Success stories showcase
- Peer-to-peer support system

## Technology Stack

### Frontend
- React with TypeScript
- Material UI for dark mode design
- Mobile-first responsive layout

### Backend
- Spring Boot REST API
- JWT authentication
- PostgreSQL database

### Deployment
- Docker containerization
- Nginx for frontend serving
